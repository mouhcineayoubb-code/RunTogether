# RunTogether 🏃‍♂️

## 📖 Project Overview
RunTogether is a full-stack web platform designed to facilitate collective running sessions. It enables users to discover local running groups, track performance, and coordinate meetups.

## 🛠️ Tech Stack
- **Frontend:** React.js, Leaflet.js
- **Backend:** Node.js, Express
- **Database:** MySQL

## 🚀 Getting Started
